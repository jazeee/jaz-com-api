# JazCom API

JazCom API

## Setup

- `npm install`

## Build

- `npm run build` or  `npm run build:watch`

## Running

Once built

- `npm start` or `node dist/main.js`

## Creating an alias

You can run this build using an alias like:

```
alias your-app-name='node ~/path-to-code/dist/main.js'
your-app-name
```
